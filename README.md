## Help

```bash
# for basic commands
make help
# for advanced commands
make help-all
```

## Building

```bash
# clone repository
git clone https://github.com/gwydirsam/DickGrayson.git
cd DickGrayson
make all
```

## Testing

```bash
make test-all
```

## Dependencies
- cmake (installed seperately)
- GNU MP library (installed seperately)
- googletest (included)
